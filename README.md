## 说明

 - 代码展示样式，将 [highlightjs][4] 制作为typecho插件形式。
 - 该项目由于我平时使用的语言比较多，生成了很多的语言JS包很大，若想要轻量级，请到[https://prismjs.com/download.html][5]自行下载。

## 使用方法

第一步：下载本插件，放在 `usr/plugins/` 目录中；

第二步：文件夹名改为RinvayCodeHighLight；

第二步：登录管理后台，激活插件；
